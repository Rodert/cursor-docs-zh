title: 自定义 API 密钥
---

### OpenAI API 密钥

Cursor 允许您输入自己的 OpenAI API 密钥，以便按照您自己的成本发送任意数量的 AI 消息。

您可以从[这里](https://platform.openai.com/account/api-keys)获取您自己的 API 密钥。要使用您自己的 API 密钥，请转到 `Cursor 设置` > `模型` > `OpenAI API 密钥` 并输入您的 API 密钥。然后，点击"验证"按钮。一旦您的密钥通过验证，您的 OpenAI API 密钥就会被启用。

### 我的 API 密钥会被存储或离开我的设备吗？

您的 API 密钥不会被存储，但每次请求都会被发送到我们的服务器。所有请求都会通过我们的后端路由，因为最终的提示构建是在那里完成的。
