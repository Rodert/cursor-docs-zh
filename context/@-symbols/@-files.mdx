### `@文件`

在 Cursor Chat 和 Cmd K 等 AI 输入框中，您可以使用 `@文件` 引用整个文件。
另外，如果您在 `@` 后继续输入，您将在 [`@代码`](/context/@-symbols/@-code) 策略后看到您的文件搜索结果。

为了确保您引用的文件是正确的文件，Cursor 会显示文件路径的预览。当您在不同文件夹中有多个同名文件时，这特别有用。

### 聊天中的长文件引用

在 Cursor 的聊天中，如果文件内容太长，Cursor 会将文件分成更小的块，并根据与查询的相关性重新排序。

### Cmd K 分块策略

对于 Cmd K，Cursor 也会根据内容长度以不同方式使用文件引用。

- 自动
  - 根据文件大小自动选择三种读取策略之一
- 完整文件
  - 将整个文件用作上下文。
- 大纲
  - Cursor 解析文件的大纲并将信息用作上下文。
- 分块
  - Cursor 将文件分成更小的块并选择最相关的一块。

### 拖放

您也可以从主侧边栏将文件拖放到聊天或 Cmd K 中，将它们添加为上下文。
